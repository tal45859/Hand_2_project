
# Angular Project Hand_2_project


## Description
The project Hand_2_ is intended for the sale of 2nd hand products and a management system project for the administrator and the advertiser

## Code languages
- TS
- HTML5
- Css
- SQL
- C#

## framework
- Angular
- ASP.NET Core Web API

## Data Bases
- SQL Server
